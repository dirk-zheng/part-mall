# 数码商城 (Digital Mall) - 产品规范

## 1. 概念与愿景

一个清新明亮的数码产品在线商城，专为年轻科技爱好者打造。界面以纯净的浅色为基调，配合科技蓝的强调色和精致的卡片设计，营造出专业而现代的购物体验。整体风格融合现代极简美学与科技感，让用户在浏览商品时感受到清新舒适的数字化体验。

## 2. 设计语言

### 美学方向
浅色科技风 + 现代极简：明亮的背景、纯净的白色卡片、科技蓝的强调色、柔和的阴影效果

### 配色方案
- **主色 (Primary)**: `#0066ff` (科技蓝)
- **次色 (Secondary)**: `#6366f1` (电紫)
- **强调色 (Accent)**: `#00d4aa` (清新绿)
- **背景色 (Background)**: `#f8fafc` (极浅灰) → `#e2e8f0` (浅灰) 渐变
- **表面色 (Surface)**: `#ffffff` (纯白)
- **文字主色**: `#1e293b` (深灰蓝)
- **文字次色**: `#64748b` (中灰)
- **边框色**: `#e2e8f0` (浅灰边框)

### 字体
- 标题: `Space Grotesk`, `Inter`, sans-serif
- 正文: `Inter`, `system-ui`, sans-serif
- 代码/价格: `JetBrains Mono`, monospace

### 空间系统
- 基础间距: 4px
- 组件间距: 16px / 24px / 32px
- 卡片圆角: 16px
- 按钮圆角: 8px / 全圆角

### 动效哲学
- 页面进入: 元素从下方淡入上移，stagger 100ms
- 悬停反馈: translateY(-4px), box-shadow 增强，300ms ease
- 按钮点击: scale 效果，50ms
- 卡片悬浮: 轻微上浮 + 阴影加深
- 页面切换: 淡入淡出 300ms

### 视觉资产
- 图标库: Lucide React
- 商品图片: Unsplash 科技/数码类
- 装饰元素: 柔和渐变背景、淡雅光晕效果

## 4. 功能与交互

### 4.1 商城列表 (首页)
- 商品卡片网格展示，每个卡片包含：
  - 商品图片 (hover 时微微放大)
  - 商品名称
  - 价格 (醒目显示)
  - 分类标签
  - "加入购物车" 按钮
- 分类筛选：手机、电脑、配件、智能穿戴、音频
- 搜索框：实时搜索商品
- 价格排序：升序/降序
- 统计卡片：商品总数、当前展示数

**交互细节**：
- 悬停卡片：阴影加深，轻微上浮
- 点击"加入购物车"：按钮变为对勾，2秒后恢复
- 分类按钮：激活态有背景色

### 4.2 购物车
- 商品列表：图片、名称、数量控制、价格
- 数量控制：+/- 按钮，最小 1
- 删除商品：点击删除图标
- 价格汇总：商品总价 + 运费估算
- 清空购物车：确认对话框
- 结账按钮：点击显示成功提示（模拟）

**状态处理**：
- 空购物车：显示插图 + "去逛逛" 链接

### 4.3 商品上架管理
- 商品列表表格：图片、名称、分类、价格、库存、操作
- 添加商品：
  - 表单字段：名称、分类、价格、库存、描述、图片 URL
  - 表单验证：必填项、价格 > 0、库存 ≥ 0
- 编辑商品：点击行打开编辑弹窗
- 删除商品：确认对话框
- 分类统计卡片：各类商品数量

**验证规则**：
- 商品名称：2-50 字符
- 价格：正数，最多两位小数
- 库存：非负整数

### 4.4 客服系统
基于 CodeBuddy SDK 的智能客服聊天：

**核心功能**：
- 实时聊天界面（模拟 AI 回复）
- 预设快捷问题按钮
- 消息时间戳
- 打字指示器动画
- 对话历史记录

**AI 回复逻辑**（基于关键词匹配）：
- 询问价格 → 回复价格相关话术 + 引导购买
- 询问物流 → 回复物流信息
- 投诉建议 → 感谢反馈
- 闲聊 → 友好互动

**交互细节**：
- 发送消息：回车或点击发送按钮
- 空消息不可发送
- 新消息自动滚动到底部
- 欢迎消息：首次打开时显示

## 5. 组件清单

### Header
- Logo + 标题（深色文字）
- 导航链接（当前页高亮为蓝色背景）
- 购物车图标（显示数量徽章）
- 用户头像
- **状态**：默认（白色玻璃背景）

### ProductCard
- 商品图片（16:9 比例）
- 分类标签（彩色胶囊 - 浅色背景）
- 商品名称（单行截断）
- 价格（主色蓝色，醒目）
- 购物车按钮（阴影效果）
- **状态**：默认、悬停（上浮+阴影）、已加入购物车、售罄

### CartItem
- 商品缩略图
- 名称和规格
- 数量调节器
- 小计价格
- 删除按钮
- **状态**：默认、删除中

### ProductForm（弹窗）
- 输入字段组
- 图片预览
- 分类下拉选择
- 价格/库存数字输入
- 取消/提交按钮
- **状态**：编辑模式、添加模式、验证错误、提交中

### ChatBubble
- 头像
- 消息内容
- 时间戳
- **状态**：用户消息（右对齐蓝色气泡）、AI 消息（左对齐白色气泡）

### ChatInput
- 文本输入框
- 发送按钮
- **状态**：默认、聚焦、发送中

## 6. 技术方案

### 技术栈
- **框架**: React 18 + Vite
- **样式**: Tailwind CSS
- **路由**: React Router v6
- **状态管理**: React Context + useReducer
- **图标**: Lucide React

### 项目结构
```
src/
├── components/     # 可复用组件
├── pages/          # 页面组件
├── context/        # 全局状态
├── data/           # 模拟数据
├── App.jsx
├── main.jsx
└── index.css
```

### 数据模型
```typescript
// 商品
interface Product {
  id: string;
  name: string;
  category: 'phone' | 'computer' | 'accessory' | 'wearable' | 'audio';
  price: number;
  stock: number;
  image: string;
  description: string;
}

// 购物车项
interface CartItem {
  product: Product;
  quantity: number;
}

// 聊天消息
interface ChatMessage {
  id: string;
  sender: 'user' | 'ai';
  content: string;
  timestamp: Date;
}
```

### 本地存储
- 购物车数据：localStorage
- 用户偏好：localStorage
- 对话历史：sessionStorage

### CodeBuddy SDK 集成
客服系统采用 CodeBuddy SDK 的设计风格，实现智能客服对话功能。
